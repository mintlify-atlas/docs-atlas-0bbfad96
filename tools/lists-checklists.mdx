Lists & Checklists is Chapter's organisation system that helps UK students track their university choices, accommodation options, job applications, and important tasks throughout their application journey.

## Overview

Create lists to save and organise anything:
- **Course Shortlists** - Track your top 5 UCAS choices and backups
- **Accommodation Lists** - Compare student housing options
- **Job & Apprenticeship Lists** - Manage applications and opportunities
- **Custom Checklists** - Task lists for any purpose
- **Template Checklists** - Pre-made lists for common student needs

<Info>
Lists sync across devices and can be shared with friends or family for collaborative decision-making.
</Info>

## Types of Lists

### Course Lists

Manage your university applications:

**UCAS Choice Types:**
- **Firm Choice** (1 only) - Your first choice if you meet the grades
- **Insurance Choice** (1 only) - Your backup if you miss firm choice grades  
- **Additional Choices** (up to 3) - Extra options through UCAS Extra or Clearing
- **Considering** - Courses you're researching

**Features:**
- Chapter enforces UCAS limits (1 firm, 1 insurance, 3 additional)
- Quick-change choice type with one tap
- Update offer status (Offer Received, Conditional, Unconditional, Rejected)
- View all details without leaving the list
- Compare courses directly from your shortlist

**Offer Holder Hub:**
Once you have offers, access the Offer Holder Hub:
- Track offer conditions
- See what grades you need
- Countdown to results day
- Decision timeline and deadlines
- Recommended checklists for offer holders

<Note>
Course lists automatically appear in the Offer Holder Hub when you mark a course as having an offer.
</Note>

### Accommodation Lists

Compare and track student housing:

**Information saved:**
- Weekly/monthly rent
- Bills included status
- Room type (ensuite, studio, shared)
- Location and distance to campus
- Contract length
- Amenities and facilities
- Provider contact details

**Actions available:**
- Add notes and ratings
- Mark as viewed
- Set viewing appointments
- Share with parents/friends
- Compare side-by-side
- Calculate total year cost

### Job & Apprenticeship Lists

Manage career applications:

**Track:**
- Job title and employer
- Salary and benefits
- Application deadline
- Application status
- Interview dates
- Custom notes

**Application Statuses:**
- Interested - Want to apply
- Applied - Submitted
- Interview Scheduled
- Offer Received
- Accepted
- Rejected

**Features:**
- Deadline reminders
- Application progress tracking
- Sort by salary, deadline, or status
- Export as CV tracking spreadsheet

### Custom Checklists

Create task lists for any purpose:

**Supported item types:**
- ☐ **Checkboxes** - Standard to-do items
- • **Bullet points** - Notes without completion tracking
- 1. **Numbered lists** - Sequential steps
- **Plain text** - Headings and descriptions

**Formatting:**
- Indent items (up to 3 levels)
- Bold and italic text
- Strikethrough completed items
- Add headings to organise sections

**Progress tracking:**
- Completion percentage (X/Y tasks done)
- Visual progress bar
- Due dates and reminders
- Overdue highlighting

<Tip>
Use the style picker (bottom toolbar) to quickly change item types. Press return to create new items.
</Tip>

## Creating Lists

<Tabs>
  <Tab title="Course List">
    **Quick add from course pages:**
    1. Browse or search for a course
    2. Tap the bookmark icon
    3. Select list or create new
    4. Choose choice type (firm, insurance, etc.)
    
    **Bulk add:**
    - Use comparison tool to add multiple courses at once
    - Import from UCAS Track (coming soon)
  </Tab>

  <Tab title="Accommodation List">
    **From accommodation search:**
    1. Find accommodation you like
    2. Tap "Add to list"
    3. Select existing list or create new
    4. Add viewing notes
    
    **Manual add:**
    - Create list first
    - Use "+" button to add properties
    - Search Chapter's accommodation database
  </Tab>

  <Tab title="Custom Checklist">
    **From scratch:**
    1. Go to Lists tab
    2. Tap "+" (Create List)
    3. Choose "Checklist"
    4. Add title and optional description
    5. Start adding tasks
    
    **From template:**
    1. Browse template library
    2. Preview template
    3. Tap "Copy to My Lists"
    4. Customise as needed
  </Tab>

  <Tab title="Job List">
    **From job search:**
    1. Find apprenticeship or job
    2. Tap "Save"
    3. Add to existing list or create new
    4. Set application deadline
    
    Lists auto-sort by deadline.
  </Tab>
</Tabs>

## Tips for Effective Lists

<AccordionGroup>
  <Accordion title="Use choice types correctly">
    Only mark courses as firm/insurance when you've actually made those decisions after receiving offers. Use "Considering" while researching.
  </Accordion>

  <Accordion title="Add notes immediately">
    When you add accommodation or a job, write down why you liked it. You'll forget after viewing 10+ options.
  </Accordion>

  <Accordion title="Set realistic deadlines">
    For checklists, set due dates a few days before actual deadlines to give yourself buffer time.
  </Accordion>

  <Accordion title="Review weekly">
    Check your lists once a week to update statuses, remove options you've ruled out, and add new ones.
  </Accordion>

  <Accordion title="Don't over-organise">
    You don't need separate lists for every category. One "Shortlist" for courses is usually enough.
  </Accordion>

  <Accordion title="Use templates as starting points">
    Copy template checklists then customise for your situation rather than starting from scratch.
  </Accordion>
</AccordionGroup>
