## Joining Societies

### Join Process

**1. Browse and Discover**
Find societies that interest you.

**2. View Profile**
Tap to see full details, events, and member info.

**3. Join Group Chat**
Tap "Join Group" to:
- Access the society's group chat
- See pinned messages with important info
- View upcoming events
- Get notifications for announcements

**4. Auto-Follow**
Joining the group automatically:
- Adds society events to your calendar
- Shows society posts in your feed
- Subscribes you to updates

<Note>
Joining the group chat is free and instant. Most societies also require Students' Union membership (usually £5-15/year) for official membership benefits and event access.
</Note>

### Official Membership

Many societies require SU membership for:
- Discounted event tickets
- Competition eligibility
- Voting rights in society meetings
- Access to exclusive resources

Purchase via:
- Students' Union website (linked from society profile)
- Society taster events
- Freshers' Fair stalls

### Claim Your Society

If you're on a society committee:

1. **Navigate to society profile**
2. **Tap "Claim This Society"**
3. **Verify your position:**
   - Upload proof (SU portal screenshot, committee email)
   - Provide Student ID
   - SU staff verification
4. **Await approval** (usually 24-48 hours)
5. **Gain admin access** to group and posting privileges

<Info>
Only verified committee members can claim and manage society pages to prevent impersonation.
</Info>

## Best Practices

<AccordionGroup>
  <Accordion title="Join During Freshers' Week">
    Most societies have taster events and sign-up deals during the first weeks. Don't miss out!

  <Accordion title="Try Before You Commit">

  <Accordion title="Don't Overcommit">

  </Accordion>

  </Accordion>

## Technical Details

### Society Data Model

```swift
struct Society {
    let id: Int
    let name: String
    let description: String?
    let universityName: UniversityName
    let imageURL: String?
    let website: String?
    let email: String?
    let link: String // SU page
    let category: SocietyCategory
    let teams: [SportsTeam]? // For sports societies
    let socialMedia: SocialMediaLinks
}
```

### Sports Team Model

```swift
struct SportsTeam {
    let id: UUID
    let name: String
    let sport: Sports
    let logo: String?
    let league: League
    let universityName: UniversityName
}
```
