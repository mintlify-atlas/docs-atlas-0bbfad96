Course Discovery is Chapter's powerful course search engine that helps students find the perfect university program from over 28,000 courses across 140+ UK universities.

## Search Interface

The course discovery system is accessible through the What's On tab and provides multiple search methods:

### Quick Search

Tap the search bar to instantly filter by:
- Course name or keyword
- University name
- Subject category
- Location/city

<Tip>
  Search is intelligent and recognizes abbreviations (e.g., "CS" for Computer Science) and common course variations.
</Tip>

## Advanced Filters

Refine your search with comprehensive filtering options:

### Entry Requirements

<Tabs>
  <Tab title="UCAS Points">
    Set minimum and maximum UCAS tariff points to match courses within your grade range.
  </Tab>
  <Tab title="A-Level Grades">
    Filter by specific A-Level requirements (e.g., AAB, BBC)
  </Tab>
  <Tab title="Qualifications">
    Select accepted qualifications: A-Levels, BTEC, IB, Scottish Highers, Access Courses
  </Tab>
</Tabs>

### Location & Campus

- **City Filter**: Select multiple cities to find courses in specific locations
- **Campus Type**: Main campus, city center, satellite campuses
- **Distance**: Set maximum distance from your current location
- **Region**: England, Scotland, Wales, Northern Ireland

### Course Details

<AccordionGroup>
  <Accordion title="Study Mode">
    - Full-time
    - Part-time
    - Sandwich year
    - Year abroad options
  </Accordion>
  <Accordion title="Course Length">
    - 1 year (postgraduate)
    - 3 years (standard degree)
    - 4 years (integrated master's, Scotland)
    - 5+ years (Medicine, Architecture)
  </Accordion>
  <Accordion title="Degree Level">
    - Undergraduate
    - Postgraduate Taught
    - Postgraduate Research
    - Foundation Year
  </Accordion>
  <Accordion title="Accreditation">
    Filter by professional body accreditation (e.g., RICS, BCS, GDC)
  </Accordion>
</AccordionGroup>

### Cost & Finance

- **Tuition Fees**: Set maximum annual tuition budget
- **Total Cost**: Include accommodation and living expenses
- **Scholarship Availability**: Show only courses with scholarships
- **Home/International**: Filter by student fee status

<Note>
  Cost estimates combine official university tuition fees with Chapter's accommodation data for accurate total cost projections.
</Note>

## Course Details View

Tapping a course reveals comprehensive information:

### Overview Section

- Course title and UCAS code
- University logo and name
- Match percentage badge (if using Course Match)
- Key stats carousel (UCAS points, fees, duration, start date)

### Entry Requirements

Detailed breakdown of:
- Typical offers (A-Levels, BTEC, IB)
- Subject-specific requirements
- English language requirements (for international students)
- Additional tests (e.g., UCAT, LNAT, portfolio)

### Course Content

<Tabs>
  <Tab title="Modules">
    Year-by-year breakdown of core and optional modules with descriptions
  </Tab>
  <Tab title="Teaching Methods">
    Lecture hours, seminars, lab work, independent study breakdown
  </Tab>
  <Tab title="Assessment">
    Exams vs. coursework split, dissertation requirements
  </Tab>
</Tabs>

### Career Prospects

Data-driven insights including:
- **Graduate Outcomes**: Employment rates, further study percentages
- **Average Salary**: Starting salaries by sector
- **Top Employers**: Companies that recruit from this course
- **Career Paths**: Common job titles and industries

<Info>
  Career data is sourced from HESA Graduate Outcomes survey and updated annually.
</Info>

### Student Ratings

Peer feedback from current students:
- Overall satisfaction rating
- Teaching quality score
- Course organization rating
- Student support rating
- Detailed written reviews

### Campus Information

If the university has multiple campuses:
- Campus name and location
- Distance from city center
- Campus facilities
- View on map (integrates with What's On map)

## Map Integration

### Campus View

Courses are displayed on an interactive map showing:
- University campus locations
- Nearby accommodation options
- City amenities (transport, shops, nightlife)

<Steps>
  <Step title="Enable Map View">
    Tap the map button at the bottom of the course search screen
  </Step>
  <Step title="Explore Locations">
    Zoom and pan to see courses in different regions
  </Step>
  <Step title="Filter by City">
    Tap city markers to filter courses in that location
  </Step>

## Course Match Integration

When you complete the Course Match quiz, search results display:

- **Match Percentage**: How well each course aligns with your profile
- **Match Badges**: Visual indicators (Excellent, Good, Fair match)
- **Why This Match**: Explanation of compatibility factors

<Warning>
  Match percentages are calculated using your Course Match responses. Update your profile to refresh recommendations.
</Warning>

## Recent & Recommended

### Recently Viewed Courses

Automatically tracks up to 100 recently viewed courses:
- Synced across devices via Supabase
- Accessible from Hub carousel
- Searchable within recent history

### Personalized Recommendations

Course suggestions based on:
- Courses in your shortlists
- Viewing history patterns
- Course Match quiz responses
- Similar user preferences (collaborative filtering)

Recommendations are grouped by category (e.g., "More Business courses", "Similar to Law at Oxford").

## Search Performance

<Note>
  Course search uses PostgreSQL full-text search with intelligent ranking. Results typically load in under 300ms with pagination support for smooth scrolling.
</Note>

### Optimization Tips

- Use filters to narrow results before scrolling
- Search syncs with What's On manager for cross-feature consistency
- Debounced search (500ms) prevents excessive API calls while typing
