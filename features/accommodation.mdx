Chapter's Accommodation Search helps students find the perfect place to live with comprehensive listings of university halls, private student accommodation, and third-party properties across the UK.

## Overview

Search 2,500+ verified student properties with:
- University-owned halls of residence
- Purpose-built student accommodation (PBSA)
- Private landlord properties
- Real-time availability and pricing

## Search Interface

The accommodation search features a hybrid map-first and list view:

### Map View

Interactive map showing:
- Property markers with price badges
- Campus location pins
- City center indicators
- Distance radius overlays

<Steps>
  <Step title="Pan & Zoom">
    Explore different areas by dragging the map and pinching to zoom
  </Step>
  <Step title="Tap Properties">
    Tap markers to see quick property cards
  </Step>
  <Step title="Filter by Location">
    Tap city markers to filter properties in that area
  </Step>
</Steps>

<Tip>
  Switch between standard, satellite, and hybrid map styles using the controls in the top-right corner.
</Tip>

### List View

Scrollable property cards showing:
- Photo carousel (swipe to view all images)
- Property name and provider
- Room type and bathroom type
- Price per week
- Distance to campus/city center
- Key amenities icons

## Draggable Search Sheet

Airbnb-style interface with three positions:

<Steps>
  <Step title="Collapsed">
    Shows property count and search bar. Map fully visible.
  </Step>
  <Step title="Half-Sheet">
    Scrollable property list with map visible at top.
  </Step>
  <Step title="Full-Screen">
    Property list takes over screen. Map hidden.
  </Step>
</Steps>

Drag the handle to adjust sheet height, or tap the Map button to collapse.

<Note>
  Sheet position syncs with map zoom: expanding the sheet zooms out the map for broader area context.
</Note>

## Search Performance

Optimized for speed:
- **Initial Load**: Properties load in batches of 20
- **Infinite Scroll**: Automatic pagination as you scroll
- **Map Load**: Separate data fetch for map (3,000+ properties) cached after first load
- **Filters**: Debounced to prevent excessive API calls (1-second delay after filter change)

<Info>
  Filter count badge shows how many active filters you have applied. Tap "Clear All" to reset.
</Info>
